# CDC Next boiler plate

## Introdução

Boiler plate para projetos CDC em Next JS.

## Principais Bibliotecas

### Gerenciamento de estado

##### React query

React Query é uma biblioteca de busca de dados e gerenciamento de estado para aplicativos React que simplifica a busca, o armazenamento em cache e a atualização de dados.

[Documentação React Query](https://tanstack.com/query/latest/docs/framework/react/overview)

##### Zustand

O Zustand **é uma biblioteca minimalista e prática para gerenciamento de estados no React e React Native**. Escolhido pela sua simplicidade e por não exigir o uso dos tradicionais Reducers do Redux, o que facilita bastante a gestão de estados sem muita complexidade.

[Documentação Zustand](https://github.com/pmndrs/zustand)

### Estilização

##### Styled Components

Styled-components é **uma biblioteca que permite escrever CSS em JS enquanto construindo componentes personalizados em Reactjs**.

##### Material UI

Material UI é uma biblioteca de componentes React de código aberto que implementa o Material Design do Google.

### Documentação

##### Storybook

Storybook é um workshop de front-end para a construção de componentes e páginas de UI isoladamente. Utilizado para desenvolvimento, teste e documentação de UI.

##### Nextra

Nextra é uma **estrutura de geração de sites simples, poderosa e flexível** que combina os melhores recursos do Next.js com recursos adicionais para a criação de sites focados em conteúdo.

### Testes

##### Jest

Jest é um poderoso Framework de Testes em JavaScript com um foco na simplicidade.
