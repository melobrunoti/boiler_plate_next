# Iniciando o projeto

### Instalando as dependências

```sh copy
npm install
```

### Rodando o projeto em modo de desenvolvimento

```sh copy
npm run dev
```

### Build

```sh copy
npm run build
```

Depois do build para rodar a versão de produção.

```sh copy
npm run start
```

## Docker

### Rodando o projeto no docker em modo de desenvolvimento

```sh copy
docker-compose --build
```

### Rodando o projeto no docker em modo de produção

```sh copy
docker-compose -f docker-compose-prod.yml up --build
```
